## Sanity Studio

Run the following command if you want to host your studio on Sanity:

<Tabs>
  <TabItem label="pnpm" icon="pnpm">
    <Code lang="sh" code={`pnpm run sanity:deploy`} />
  </TabItem>
  <TabItem label="yarn" icon="seti:yarn">
    <Code lang="sh" code={`yarn run sanity:deploy`} />
  </TabItem>
  <TabItem label="npm" icon="seti:npm">
    <Code lang="sh" code={`npm run sanity:deploy`} />
  </TabItem>
</Tabs>

More info about [deploying Sanity](https://www.sanity.io/docs/deployment).
