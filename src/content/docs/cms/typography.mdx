You can add custom fonts to your theme.

<Steps>
1. Use a font library such as [Google Fonts](https://fonts.google.com/) or
[Fontshare](https://www.fontshare.com/) to download font families, styles and
variables.

2. Add the font family to Heading or Body<br/>![Font
family](../../../assets/font_family.png)

3. Upload the font assets (use .woff2 files for better performance) and choose font style and font weight<br/>![Font assets](../../../assets/font_assets.png)

4. Update font base size, letter spacing and line height<br/>![Font
settings](../../../assets/font_settings.png)

</Steps>
